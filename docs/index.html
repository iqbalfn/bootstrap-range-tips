Range Tips

Add informative tooltips to exists bootstrap custom range slider.

About

Bootstrap 4.3.1 already has built-in custom range slider which is by default already nice. It just miss the information user currently selected value. And this is the simple plugins to add custom tooltips to the element to show user the value they currently select.

Please note that the tooltips element use bootstrap tooltips style. So the element should be include in your custom build.

Another note is that the position for root element font size is 16px. I'll need more time to provide other root font size.
